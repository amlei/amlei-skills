# amlei-story-generator

把任意主题写成有钩子、有故事、有共鸣的长文。支持六类素材：书 / 人 / 事 / 物 / 理 / 案。

## 触发方式

提及"生成故事/写个故事/故事化/讲讲 X 的故事/拆书/story"自动触发，或调用 `/amlei-story-generator`。

## 功能

| 功能 | 说明 |
|------|------|
| **书** | 作者背景 → 核心论点 → 内容展开 → 金句定点投放 → 落回读者 |
| **人** | 决定性瞬间 → 来路 → 关键抉择与代价 → 收获与启发 |
| **事** | 定格瞬间 → 为何发生 → 连锁反应 → 暗线 → 今天的回响 |
| **物** | 反直觉现象 → 诞生过程 → 演进竞争 → 改变了什么 → 边界与代价 |
| **理** | 困惑现象 → 概念诞生 → 例子让你懂 → 意味着什么 → 边界 |
| **案** | 还原现场 → 反常之处 → 拆解机制 → 提炼洞察 → 读者怎么用 |

## 工作流

1. **取材** — 先联网搜索核实素材（人名/年代/数据/金句必须搜到再写）
2. **起稿** — 钩子开篇 → 3–7 个序号小节（钩子式标题）→ 场景先行 → 金句定点
3. **打磨** — 通读自查三步法则
4. **送评** — 独立评判者 agent 按 rubric 逐维打分，达标线 90 分，最多 3 轮评分

## 铁律

- 金句必须真实可溯源，搜不到原文绝不引用
- 只用一个 `#`（全文标题），小节用 `## 01 / 02…`
- 长度 2000–4000 字
- 非说明文、非百科式开篇

## 输出

默认对话内输出；存盘时写入 `故事-<主题>.md`。
