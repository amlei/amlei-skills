# 评判标准（量化）

评判者按此表逐维打分，总分 100。**达标线：总分 ≥ 90，且全部硬性门槛通过。**

## 评分表

| 维度 | 满分 | 打分规则 |
|---|---|---|
| 钩子开篇 | 15 | 开篇前 1–2 句存在明确认知缺口 / 反常识 / 提问。是 = 15，否 = 0。 |
| 标题 | 10 | 标题承诺一个具体认知且正文兑现（非泛泛、非"主题 + 简介"式）。是 = 10，否 = 0。 |
| 序号钩子标题 | 15 | 钩子式小节标题占比 ≥ 80%。得分 = 占比 × 15（四舍五入）。 |
| 场景先于术语 | 15 | 核心概念中"先场景后命名"占比 ≥ 80%。得分 = 占比 × 15。 |
| 金句 | 20 | ① 每条可溯源真实；② 定点投放（高潮 / 转折 / 结尾）；③ 每节 ≤ 2 条。三项各约 1/3。 |
| 类型骨架 | 10 | 用了对应素材类型（书 / 人 / 事 / 物 / 理 / 案）的拆解骨架。是 = 10，否 = 0。 |
| 要素齐全 | 5 | 背景 / 主线 / 血肉 / 落点 / 金句 五项齐全（书类对应 作者 / 简介 / 核心内容 / 观点 / 金句）。每缺一项 −1。 |
| 归于共鸣 | 10 | 结尾回扣开篇 + 接回读者生活，且无禁用收尾腔调。有禁用腔调 = 0。 |

## 硬性门槛（任一不过 → 直接不达标，与总分无关）
- 🚫 金句编造（最高红线）
- 🚫 要素不齐（背景 / 主线 / 血肉 / 落点 / 金句 缺一不可）
- 🚫 禁用收尾腔调（"总而言之""希望对你有帮助"等）
- 🚫 说明文开篇（百科式 / "X 是……的……"定义起手）

## 评判者输出格式
1. 逐维得分表（维度 / 得分 / 一句依据）
2. 总分
3. 判定：达标 / 不达标（附触发的不达标项）
4. 不达标时：按优先级列出具体修改点，定位到小节或句子
