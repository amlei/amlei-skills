# 按素材类型 · 拆解骨架

素材类型决定正文长什么样。**先判类型，再取骨架。** 一本书和一个人、一件事，讲法完全不同。

每类给两件：**推荐叙事弧 / 易犯错误**。

---

## 0 · 书
- **叙事弧**：一个痛点 / 问题 → 作者的答案 → 展开论证 → 落回读者。
- **易犯**：说明文腔；只罗列要点无叙事。
- 书的子类（方法论 / 经管 / 心理…）见文末「书 · 八大子类」。

## 1 · 人（人物 / 传记 / 访谈）
- **叙事弧**：一个决定性瞬间 → 这个人从哪来 → 关键抉择及其代价 → 收获 → 给读者的启发。
- **易犯**：年表流水账；只见成就不见代价；造神。

## 2 · 事（事件 / 新闻 / 历史片段）
- **叙事弧**：定格一个瞬间 → 它为何发生 → 连锁反应 → 表面下的暗线 → 今天的回响。
- **易犯**：编年流水账；因果不清；只讲经过不讲意义。

## 3 · 物（产品 / 技术 / 公司 / 品牌）
- **叙事弧**：一个反直觉现象 / 痛点 → 它怎么被造出来 → 演进与竞争 → 改变了什么 → 边界 / 代价。
- **易犯**：变成产品说明书 / 软文；只吹不批；无人物。

## 4 · 理（概念 / 理论 / 方法 / 趋势）
- **叙事弧**：一个让人困惑的现象 → 这个概念怎么命名 / 诞生 → 用例子让你懂 → 它意味着什么 → 边界。
- **易犯**：定义先行不接地气；堆术语；不接生活。

## 5 · 案（案例 / 研究 / 数据 / 经历）
- **叙事弧**：还原案例现场 → 反常之处 → 拆解机制 → 提炼洞察 → 读者怎么用。
- **易犯**：只讲故事不提炼；结论过度推广；缺数据支撑。

---

## 跨类型怎么办
一个素材常跨类（如某公司的崛起 = 物 + 人 + 事）。**取主类骨架，借副类的强元素。** 主类定骨架，副类加料。

---

## 附：书 · 八大子类（仅当素材是"书"时，按子类细化）

### 方法论（自助 / 效率 / 习惯）
代表：《原子习惯》《深度工作》
- 要素：研究 / 实验、方法为何有效的机制、步骤拆解、反例与误区、可执行清单。
- 弧：痛点 → 机制 → 方法怎么被想出 → 步骤落地 → 常踩的坑。
- 易犯：写成说明书；只给"怎么做"不给"为什么"。

### 经管 / 商业
代表：《原则》《从 0 到 1》
- 要素：真实案例、决策时刻、思维链条(A→B→C)、行业洞察、模型 / 框架。
- 弧：反直觉现象 → 案例 → 拆思维链 → 提炼模型 → 模型边界。
- 易犯：只罗列金句式结论，不展示推理。

### 心理学
代表：《思考，快与慢》《影响力》
- 要素：经典实验、理论命名、认知偏差、日常投射、如何对抗。
- 弧：还原实验 → 震惊结果 → 命名 → 投射日常 → 对抗 / 利用。
- 易犯：术语先行，实验压成一句结论。

### 历史
代表：《人类简史》《万历十五年》
- 要素：关键转折、因果链、人物、长期模式、与今天的回响。
- 弧：定格瞬间 → 为何发生 → 连锁反应 → 暗线 → 今天的回响。
- 易犯：编年流水账；因果不清。

### 传记 / 回忆录
代表：《埃隆·马斯克传》《鞋狗》
- 要素：决定性瞬间、性格成因、关键抉择、代价、可借鉴处。
- 弧：决定性瞬间 → 来路 → 抉择与代价 → 收获 → 启发。
- 易犯：变年表；只见成就不见代价；造神。

### 小说 / 文学
代表：《百年孤独》《活着》
- 要素：人物困境、冲突升级、主题浮现、象征 / 意象、命运感。
- 弧：人物两难 → 冲突升级 → 主题浮现 → 意象 / 金句 → 共鸣。
- 注意：可适度剧透，但保留关键悬念张力。

### 科普
代表：《时间简史》《自私的基因》
- 要素：反直觉事实、发现过程、强类比、未解之谜、意味着什么。
- 弧：反直觉事实 → 怎么发现 → 类比让你懂 → 翻转世界观 → 边界。
- 易犯：类比失真；堆术语。

### 哲学 / 思想
代表：《被讨厌的勇气》《西西弗神话》
- 要素：刺人的问题、前人的回答、本书颠覆、思想实验、活法改变。
- 弧：刺人问题 → 原本怎么答 → 本书怎么掀桌 → 思想实验 → 活法不同。
- 易犯：掉书袋；不接生活。
