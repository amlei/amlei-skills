# 简历撰写助手

当用户提到"简历"、"工作经历"、"项目经历"等关键词时使用。处理简历相关的所有任务，包括撰写、优化、整理工作/实习经历、项目经历和专业技能，基于用户提供的项目数据、文档或代码库生成针对目标岗位的简历内容。

## 参数
- `$ARGUMENTS[0]`：目标岗位/职位方向（必需）
- `$ARGUMENTS[1]`：数据文件或目录路径（可选，默认为当前工作目录）

## 工作流程

1. **数据收集**：使用 `Grep` 搜索关键词（技术栈、框架、工具、项目相关词汇），用 `Glob` 查找文档，用 `Read` 阅读项目详情
2. **制定计划**：识别与目标岗位相关的项目，确定时间顺序，提取量化成果
3. **撰写内容**（遵循 STAR 原则：情境-任务-行动-结果）：
   - 工作/实习经历（最新在前，突出职责、内容、量化成果）
   - 项目经历（突出项目内容、工作内容、项目成果）
   - 专业技能（技术栈 + 核心能力 + 可选的证书/语言）
4. **输出确认**：生成简历并询问是否需要调整
